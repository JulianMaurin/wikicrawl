# Quick start

## Run the docker compose instance locally

### Requirement

- `docker`,
- `docker-compose`
- `make`

__Nb: Managing docker as a non-root user is required (see: [Docker post-installation steps for Linux](https://docs.docker.com/engine/install/linux-postinstall/)).__
